# Hostragons-WHMCS-Domain-Reseller

## Overview

It is an open source software that allows you to sell Domain Names on Hostragons for WHMCS.

## Features

* Domain name registration.
* Domain name renewal.
* Domain name transfer
* Whois settings
* Detailed statistics.
* User friendly interface.
* Support PHP 5/7.
* Support WHMCS V6/7
